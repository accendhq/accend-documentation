# DEBUG.CONFIG

Returns the current API configuration visible to the custom functions runtime.

Syntax\
`=ACCEND.DEBUG.CONFIG()`

Parameters\
None.

Returns\
A JSON string showing the saved API base URL and token currently available to the custom functions runtime.

Example\
`=ACCEND.DEBUG.CONFIG()`

Notes\
This function is intended for troubleshooting configuration issues. It is not meant for normal modeling workflows.
